# timeranges [![Build Status](https://travis-ci.org/heff/timeranges.svg?branch=master)](https://travis-ci.org/heff/timeranges)

> My wondrous module


## Install

```
$ npm install --save timeranges
```


## Usage

```js
var timeranges = require('timeranges');

timeranges('unicorns');
//=> unicorns & rainbows
```
